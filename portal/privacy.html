4. Transferência Internacional de Dados

Os dados biométricos (imagens faciais e vetores) são armazenados em servidores Amazon Web Services (AWS), região us-east-1 (Virgínia do Norte, EUA). A transferência é realizada com base no Acordo de Processamento de Dados (DPA) da AWS, nos termos do Art. 33, II da LGPD. A AWS é certificada pelos frameworks ISO 27001 e SOC 2 Type II.

6. Direitos do Titular (LGPD Art. 18)

  • Confirmação e Acesso: exportar seus dados no app mobile (Perfil → "Exportar meus dados")
  • Correção: solicitar correção via e-mail ao DPO
  • Eliminação: revogar biometria no app (Perfil → Biometria → Revogar); exclusão completa via DPO
  • Portabilidade: exportar dados em formato JSON (app mobile)
  • Revogação do Consentimento: disponível a qualquer momento no app (Perfil)
  • Oposição: contestar tratamento via e-mail ao DPO

7. Segurança

O sistema adota as seguintes medidas técnicas de segurança:

  • Autenticação por JWT com expiração de 60 minutos e refresh token com rotação
  • Senhas armazenadas com bcrypt + pbkdf2_sha256
  • Comunicação via HTTPS (TLS 1.2+) em produção
  • Imagens faciais acessíveis apenas via URL pré-assinada com expiração de 300 segundos
  • Rate limiting em endpoints críticos (login: 10/min, upload biométrico: 10/min)
  • Logs de auditoria com e-mails mascarados
